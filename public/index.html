Server analyzes text files (.txt, .md, .text, .log) in a given directory.

Endpoint:

GET /analyze?dir=<path>&useWorkers=true|false

Parameters:

  • dir (required) - path to directory
  • useWorkers (optional) - true = Worker Threads, false = Async (default: true)

Examples:

  • /analyze?dir=./test-files (with Worker Threads)
  • /analyze?dir=./test-files&useWorkers=false (without Worker Threads)
